---
title: "Updates the Rating for a Completion Log"
api: "PATCH https://www.trybaseplate.io/endpoints/{id}/logs"
description:
  "Updates the rating of a completion using human feedback from your users. If
  you surface a like/dislike or other rating system, you can track the feedback
  in Baseplate."
---

## Parameters

### Path

<ParamField path="id" type="string" required>
  Baseplate endpoint ID
</ParamField>

### Header

<ParamField header="Authorization" type="string" required>
  Baseplate API key. Needs to be in the format "Bearer &BASEPLATE_API_KEY"
</ParamField>
<ParamField header="Content-Type" type="string">
  Use application/json
</ParamField>

### Body

<ParamField body="positive_rating" type="Boolean">
  True indicates a positive rating (thumbs up)
</ParamField>
<ParamField body="usage_id" type="">
  The usage ID, which is returned when calling the completions endpoint
</ParamField>
<ParamField body="logs" type="Array">
  Array of logs to batch update. Each object should contain a positive_rating
  and usage_id. Either "logs" or "positive_rating + "usage_id" is required.
</ParamField>
