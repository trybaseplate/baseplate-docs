---
title: "Upload a Document (file) to a Dataset"
api: "POST https://www.baseplate.ai/api/datasets/7129ce91-7980-4518-9f51-94fef8a6b955/upload"
description: "Upload a document directly and specify chunk size/overlap, and metadata. Limit to 100MB uploads"
---

### Path Parameters

<ParamField path="id" type="string" required>
  Dataset ID
</ParamField>

### Header

<ParamField path="Authorization" type="string" required>
  Baseplate API key. Must be in the format "Bearer $BASEPLATE_API_KEY"
</ParamField>

### Body

<ParamField body="chunk_size" type="number">
  Chunk size to chunk documents. Defaults to 1000 characters. Limited to 1000 for hybrid and 4000 for standard datasets.
</ParamField>

<ParamField body="chunk_overlap" type="number" default="150 characters">
  Overlap size for chunks. Must be smaller than chunk size.
</ParamField>

<ParamField body="file" type="file" required>
  Attach a file to the request form-data.
</ParamField>

<ParamField body="document-id" type="string" required>
  The document ID to create for this file. If none is provided, a uuidv4 ID is generated for you.
</ParamField>
